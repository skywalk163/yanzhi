# 2026-06-09

## Playground "names is not defined" Bug Fix

**问题：** 点击侧边栏"示例"时显示 "加载失败 (names is not defined)"

**根因：** 旧版 `script.js` 中 `renderExamples()` 函数使用了变量名 `const names = data.examples || []`，新版已改为 `examples`。但浏览器缓存了旧版 JS 文件，导致运行时抛出 `ReferenceError: names is not defined`。

**修复：**
1. `index.html` — 脚本引用加版本号 `?v=3` 强制缓存失效
2. `script.js` — catch 块添加 `console.error('renderExamples error:', e)` 便于调试，提示信息中加入 "Ctrl+F5 强制刷新"
3. `server.py` — Cache-Control 头增强为 `no-store, no-cache, must-revalidate, max-age=0`

**验证：** 服务器正常运行，API `/api/examples` 返回正确的示例列表。
