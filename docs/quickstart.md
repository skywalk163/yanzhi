
# 言知 (Yanzhi) 快速入门指南

## 什么是言知？

言知是一个根植于汉语思维结构的中文编程语言。它允许您使用自然的中文语法进行编程，同时保留了编程语言的强大表达能力。

## 安装

### 从源码安装

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-repo/yanzhi.git
   cd yanzhi
   ```

2. 安装依赖：
   ```bash
   pip install -e .
   ```

3. 验证安装：
   ```bash
   yanzhi --version
   ```

## 第一个言知程序

创建一个名为 `hello.yan` 的文件：

```yan
打印 "你好，世界！"。
```

运行程序：
```bash
yanzhi hello.yan
```

## 基本语法

### 变量定义

```yan
定 名字 = "张三"。
定 年龄 = 25。
```

### 数学运算

```yan
定 结果 = $(1 + 2 * 3)。
打印 结果。
```

### 条件判断

```yan
如果 年龄 大于 18 那么
  打印 "成年人"。
否则
  打印 "未成年人"。
```

### 循环

```yan
循环 数 于 [1, 2, 3, 4, 5]：
  打印 数。
```

### 函数定义

```yan
定 阶乘 = 函 n：
  如果 n 等 0 那么
    返回 1。
  否则
    返回 n 乘 阶乘 $(n - 1)。
```

## 言律句式（自然语言语法）

言知支持自然的中文句式，让代码更易读：

### "当...就..." 句式

```yan
当 温度 大于 30 就 打印 "太热了！"。
```

### "每隔...做..." 句式

```yan
每隔 5 秒 做 打印 "心跳"。
```

### "...的时候：" 作用域块

```yan
回家的时候：
  开灯。
  做饭。
  吃饭。
```

## 交互式环境

启动 REPL 交互式环境：

```bash
yanzhi -i
```

在 REPL 中直接输入言知代码：

```bash
言知> 定 x = 5。
言知> 打印 x。
=> 5
```

## 标准库

言知提供了丰富的标准库：

### 数学函数

```yan
定 平方根 = 平方根 25。
定 绝对值 = 绝对 -10。
```

### 列表操作

```yan
定 列表 = 列表 1 2 3。
定 长度 = 长度 列表。
定 排序后 = 排序 列表。
```

### 字符串处理

```yan
定 大写文本 = 大写 "hello"。
定 子字符串 = 子串 "你好世界" 0 2。
```

## 进阶特性

### 双轨逃逸机制

在言知中，您可以在中文代码中嵌入数学表达式和Python代码：

```yan
定 结果 = $(sin(π/2) + cos(0))。  # 数学表达式
定 文件列表 = {{os.listdir(".")}}。  # Python代码
```

### 宏系统

```yan
宏 重复 次数 代码：
  循环 数 于 范围 次数：
    执行 代码。

重复 3：
  打印 "你好"。
```

## 示例程序

### 计算斐波那契数列

```yan
定 斐波那契 = 函 n：
  如果 n 小于 2 那么
    返回 n。
  否则
    返回 斐波那契 $(n-1) + 斐波那契 $(n-2)。

循环 数 于 范围 10：
  打印 "斐波那契" 数 "是" 斐波那契 数。
```

### 简单的Web服务器

```yan
导入 http。

定 服务器 = http.创建服务器 函 请求 响应：
  响应.写 "你好世界"。

服务器.监听 8080。
打印 "服务器运行在 http://localhost:8080"。
```

## 常见问题

### 1. 如何调试程序？
使用 `yanzhi -d` 启动调试模式，或使用内置的调试工具。

### 2. 言知支持哪些平台？
目前支持 Windows、macOS 和 Linux。

### 3. 性能如何？
言知提供了两种执行引擎：AST 解释器（开发友好）和字节码 VM（生产性能）。
